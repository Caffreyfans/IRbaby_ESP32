## 仓库描述：
基于 esp-idf-v5.4.1 开发的万能红外遥控固件，可通过网页前端完成配置。

## 支持硬件
目前支持`ESP32`、`ESP32C6`。

**开发计划**
- [ ] 支持 OTA 升级
- [ ] 支持 MQTT 控制，接入 HomeAssistant
- [ ] 接入 HomeKit

## 拉取
```shell
git clone https://github.com/Caffreyfans/IRbaby_ESP32.git
git submodule update --init --recursive
```
